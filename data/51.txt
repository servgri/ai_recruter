Задание 3
Изначальный запрос:
SELECT DEPARTMENT_ID,
 DEPARTMENT_NAME,
 SUM(SALARY) TOTAL_SALARY 
FROM EMPLOYEE E,                                  -- EMPLOYEES
 DEPARTMENTS D                                    -- Лучше использовать JOIN явно
WHERE E.DEPARTMENT_ID = D.DEPARTMENT_ID
 AND MANAGER_ID = NULL                            -- Сравнение с NULL через = некорректно
 AND LOCATION_ID = (SELECT LOCATION_ID
 FROM LOCATIONS
 WHERE CITY = 'SEOUL')
 AND SUM(SALARY) >= 100000                        -- В WHERE нельзя делать агрегатные функции
GROUP BY DEPARTMENT_NAME                          -- ОШИБКА, в GROUP BY должны быть указаны все поля, указанные в SELECT

Правильный запрос:

SELECT DEPARTMENT_ID,
 DEPARTMENT_NAME,
 SUM(SALARY) AS TOTAL_SALARY
FROM EMPLOYEES E,
JOIN DEPARTMENTS D 
	ON E.DEPARTMENT_ID = D.DEPARTMENT_ID
JOIN LOCATION L 
	ON D.LOCATION_ID = L.LOCATION_ID
WHERE MANAGER_ID IS NULL
 AND L.CITY = 'SEOUL'
GROUP BY D.DEPARTMENT_ID, D.DEPARTMENT_NAME
HAVING SUM(E.SALARY) > 100000;
