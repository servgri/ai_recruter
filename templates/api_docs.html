Документация API

Полная документация по всем эндпоинтам API системы обработки заданий.

Базовый URL: http://localhost:5000

Форматы данных: JSON для запросов и ответов (кроме загрузки файлов - multipart/form-data)

Основные эндпоинты

POST /api/upload

Описание: Загрузка и обработка файлов

Параметры запроса:
  • files (multipart/form-data) - Файлы для загрузки (можно несколько)
Формат ответа: 
{
  "status": "success",
  "doc_id": 123,
  "filename": "example.docx",
  "message": "File uploaded, processing started"
}
GET /api/documents

Описание: Получить все документы из базы данных

Параметры запроса (query):
  • limit (optional) - Ограничение количества результатов
  • offset (optional) - Смещение для пагинации
Формат ответа: 
{
  "status": "success",
  "documents": [...],
  "count": 10
}
GET /api/report/<doc_id>

Описание: Получить HTML отчет по документу

Параметры пути:
  • doc_id (integer) - ID документа
Формат ответа: HTML страница с отчетом
POST /api/documents/<doc_id>/approve

Описание: Одобрить документ (требуется наличие итогового комментария)

Параметры пути:
  • doc_id (integer) - ID документа
Формат ответа: 
{
  "status": "success",
  "message": "Document approved"
}
POST /api/documents/<doc_id>/block

Описание: Заблокировать документ

Параметры пути:
  • doc_id (integer) - ID документа
Формат ответа: 
{
  "status": "success",
  "message": "Document blocked"
}
POST /api/documents/<doc_id>/unblock

Описание: Разблокировать документ

Параметры пути:
  • doc_id (integer) - ID документа
DELETE /api/documents/<doc_id>/delete

Описание: Удалить документ

Параметры пути:
  • doc_id (integer) - ID документа
POST /api/report/<doc_id>/save-grades

Описание: Сохранить оценки и комментарии для задания

Параметры запроса (JSON):
  • task_num (integer, 1-4) - Номер задания
  • score (float, optional) - Оценка для заданий 1-3 (0-10)
  • logic_score (float, optional) - Оценка логичности для задания 4 (0-100)
  • originality_score (float, optional) - Оценка оригинальности для задания 4 (0-100)
  • comment_student (string, optional) - Комментарий для студента
POST /api/report/<doc_id>/save-overall-impression

Описание: Сохранить итоговый комментарий (общее впечатление)

Параметры запроса (JSON):
  • overall_impression (string) - Текст итогового комментария
GET /api/statistics

Описание: Получить статистику для диаграмм

Формат ответа: 
{
  "status": "success",
  "pie_chart": {...},
  "bar_chart": {...},
  "line_chart": {...}
}
POST /api/competition/complete

Описание: Завершить конкурс, выбрав топ-N победителей из одобренных работ

Параметры запроса (JSON):
  • top_n (integer) - Количество победителей
POST /api/competition/start

Описание: Начать новый конкурс (сбросить статусы победителей)

POST /api/winners/send-messages

Описание: Отметить сообщения как отправленные победителям

POST /api/documents/batch-approve

Описание: Одобрить несколько документов

Параметры запроса (JSON):
  • doc_ids (array of integers) - Массив ID документов
POST /api/documents/batch-block

Описание: Заблокировать несколько документов

POST /api/documents/batch-unblock

Описание: Разблокировать несколько документов

POST /api/documents/batch-delete

Описание: Удалить несколько документов

Эндпоинты Blueprint'ов

POST /parser/upload

Описание: Загрузка и парсинг файла (legacy endpoint)

POST /cleaner/clean-tasks

Описание: Очистка заданий от хвостов (детекция и перераспределение)

GET /cleaner/status/<filename>

Описание: Получить статус очистки для файла

POST /embeddings/generate

Описание: Генерация эмбеддингов для документа

GET /embeddings/<filename>

Описание: Получить сохраненные эмбеддинги для файла

POST /analysis/similarity

Описание: Вычислить схожесть с эталонными ответами и существующими работами

POST /analysis/cheating-detection

Описание: Детекция читинга и использования LLM

GET /analysis/report/<filename>

Описание: Получить полный отчет анализа для файла

POST /api/reprocess/<doc_id>

Описание: Переобработать конкретный документ

POST /api/reprocess-unprocessed

Описание: Переобработать все необработанные документы

GET /health

Описание: Проверка работоспособности сервиса

GET /api/info

Описание: Получить информацию о сервисе и список эндпоинтов
