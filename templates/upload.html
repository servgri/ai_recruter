API Эндпоинты проекта

Основные эндпоинты:

  • POST /api/upload - Загрузка и обработка файлов
  • GET /api/documents - Получить все документы
  • GET /api/report/<doc_id> - Получить отчет по документу
  • POST /api/documents/<doc_id>/approve - Одобрить документ
  • POST /api/documents/<doc_id>/block - Заблокировать документ
  • POST /api/documents/<doc_id>/unblock - Разблокировать документ
  • DELETE /api/documents/<doc_id>/delete - Удалить документ
  • GET /api/statistics - Получить статистику для диаграмм
  • POST /api/competition/complete - Завершить конкурс (выбрать победителей)
  • POST /api/winners/send-messages - Отправить сообщения победителям
